## 🛠️ 开发工具

### 快速命令
```bash
# 启动开发环境
./scripts.sh deploy start

# 打开Admin后台
open http://localhost:1337/admin

# 查看自定义菜单
# 登录后台查看左侧导航栏
```

### 配置文件位置
- **Admin配置**: `backend/src/admin/app.js`
- **菜单权限**: Admin后台 → Settings → Roles & Permissions
- **内容类型**: Admin后台 → Content-Type Builder
