- **1:1设计稿还原**：像素级精确还原设计稿的视觉效果

## 🔧 主要优化内容

### 1. 标题样式优化
- **字体大小**：从30px调整为32px，完全匹配设计稿

- **居中显示**：使用 `margin: 0 auto` 和 `max-width` 确保完美居中

### 2. 居中显示优化

- **响应式居中**：在不同屏幕尺寸下保持居中对齐效果

### 3. 卡片布局优化
- **阴影效果**：调整为 `0px 4px 20px 0px rgba(0, 0, 0, 0.30)`，增强立体感

- **边框样式**：使用 `rgba(42, 42, 42, 0.70)` 的半透明边框
