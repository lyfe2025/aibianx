## 📈 **技术栈概览**

### **前端技术**
- **框架**: Next.js 14 (App Router + SSR/SSG)
- **语言**: TypeScript (100%类型安全)
- **样式**: 纯CSS + CSS变量系统
- **组件**: 原子设计系统 (159个自定义图标)

### **后端技术**
- **CMS**: Strapi 5.x (内容管理)
- **数据库**: PostgreSQL (主数据库)
- **搜索**: MeiliSearch (全文搜索)
- **邮件**: BillionMail (邮件营销)

### **开发工具**
- **版本控制**: Git + GitHub
- **包管理**: npm
- **代码规范**: ESLint + Prettier + TypeScript
- **容器化**: Docker (生产部署)
