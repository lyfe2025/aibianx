| 项目信息 | 详情 |
|---------|-----|
| **项目名称** | AI变现之路 (aibianx) |
| **项目类型** | AI内容平台 + 会员订阅系统 |

## 🚀 快速开始

### 新手入门 (5分钟上手)
```bash
# 1. 克隆项目
git clone <repository-url> && cd aibianx

# 2. 一键启动开发环境
chmod +x sh/development/quick-start.sh && ./sh/development/quick-start.sh

# 3. 浏览器访问
open http://localhost:3000
```

### 核心页面
- **[首页](http://localhost:3000)** - 品牌展示、功能介绍、3D效果
- **[AI周刊](http://localhost:3000/weekly)** - 智能搜索、分类筛选
- **[文章详情](http://localhost:3000/weekly/ai-revenue-model)** - 内容展示、相关推荐
- **[关于平台](http://localhost:3000/about)** - 平台介绍、会员特权
- **[个人中心](http://localhost:3000/profile)** - 用户管理、收藏订阅

### 🎯 核心功能特性

#### 📚 内容管理
- **智能搜索** - 支持标题、内容、标签的实时搜索
- **分类筛选** - 8种文章分类，useMemo性能优化
- **文章阅读** - Markdown渲染、代码高亮、目录导航
- **相关推荐** - 基于标签和分类的智能算法

#### 🔐 用户认证  
- **多方式注册** - 邮箱注册 + GitHub OAuth
- **安全登录** - 密码强度检测、忘记密码找回
- **会员管理** - 倒计时营销、特权展示、订阅管理
- **个人中心** - 信息管理、阅读历史、收藏功能

#### 💎 交互体验
- **响应式布局** - 4个断点适配（480px/768px/1024px/1440px）
- **微交互动画** - 悬停效果、点击反馈、状态切换
- **3D背景效果** - Three.js实现，支持路由切换重新初始化
- **毛玻璃设计** - backdrop-filter原生实现

## 📊 项目统计

### 📦 技术栈组成
```
前端架构
├── Next.js 14 (App Router)     # React全栈框架
├── React 19.1.0               # UI库
├── TypeScript 5               # 类型安全
├── 纯CSS + CSS变量            # 样式系统（已移除Tailwind）
├── Zustand 5.0.6              # 状态管理
├── Framer Motion 12.23.6      # 动画库
├── Three.js                   # 3D效果
└── Radix UI                   # 无障碍组件
```
